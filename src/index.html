Physics simulations for JS

A hobby project building some physics simulations with JS.

Simple cube rotation

Cube collisions require rotations to look realistic. For this we implement the SAT algorithm, clipping to find contact manifolds and upgrade the physics to allow for rotations.

Multiball

Using an octree to handle many game objects. The simulation gains a very measurable performance difference by heavily reducing the number of expensive collision checks.

Ball collisions

Collisions between movable spheres and immovable walls.

Collision detection

Collision detection, calculating impulses and updating the simulation.

Simulating gravity in space

Demonstration of forces on objects, and using the semi-implicit euler method to update the simulation.
